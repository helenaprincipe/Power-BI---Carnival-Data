# Power BI Project -- Carnival Analysis

This repository contains a Power BI report focused on analyzing data
related to carnival activities, performance metrics, and insights.

## Project Overview

The **Carnival Power BI Report** provides: - Visual dashboards for key
indicators\
- Data segmentation and filtering\
- Performance comparison between categories\
- Interactive insights for decision-making

## Tools Used

-   **Microsoft Power BI Desktop**
-   **Data Modeling (DAX, relationships, measures)**
-   **Visualization and Dashboard Design**

## File Structure

-   `README.md` -- Project documentation

## How to Use

1.  Download the `.pbix` file from this repository.\
2.  Open it using **Power BI Desktop**.\
3.  Interact with the dashboards, filters, and visual elements as
    needed.
